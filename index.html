About Us

This is a website template that you can use for your project. It has the following features:

  • An About us page that introduces your website and its purpose
  • A basic structure along with navigation and front-end bootstrap
  • A page for adding item list where you can enter the name, price, description and image of each item
  • A page for displaying item list where you can see the details of all the items you have added
  • A page for creating new items using the form where you can fill in the required fields and submit the data
  • A contact us page where you can send us your feedback and queries
